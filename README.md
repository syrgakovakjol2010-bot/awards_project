# 🏆 Creators Awards | Mistori Copani (M.C.)

**Creators Awards** — это интерактивная платформа для голосования за лучших стримеров и ютуберов года и десятилетия. Проект делит номинантов на два лагеря: **СНГ** и **Весь мир**.

> ⚡ **Главная цель:** Когда компания M.C. разбогатеет, победитель, занявший Топ-1, получит настоящую золотую статуэтку высшей пробы (999 карат)!

---

## 🛠️ Технологический стек
* **Backend:** Python 3 + Flask
* **Frontend:** HTML5, CSS3 (адаптивный геймерский дизайн)
* **Database:** JSON (локальное хранилище голосов)
